Rogue Pickings

Rogue Pickings is the fictional food truck that brings meals with the freshest ingredients to the neighborhood! I coded this based on a PS design comp. To accurately portray some of the deliciousness this company had to offer, I thought a gallery page was needed. All code validated for an issues free launch!

Jubilee Austen

Jubilee Austen is a fictional developer from Rhode Island who had her start in journalism before entering the exciting world of code. This project was done in partial completion of my course at Skillcrush, where I got to flex my HTML and CSS muscles by converting the Photoshop design comps to a great online portfolio for the lovely Ms. Austen. All the images and specs were extracted from the design comp in PS, coded and then validated for any possible issues.

Recipe Card

Recipe Cards are not only handy tools for creating delicious dishes, but are most times preserved for generations. Digital recipe cards are a great way to convert the aging paper or create a platform for sharing treasured recipes. I created these cards with the use of clean reusable HTML and CSS code. Feast on!
